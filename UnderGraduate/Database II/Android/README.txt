- User Manual for Android Studio Project Database2

** Requirements
1. Android Studio w/ emulator
2. localhost address 
3. XAMPP (Apache and mySQL running)
4. graduate1.php should be located within the htdocs folder within xampp folder within C drive

** Steps to Take
1. Change url link to match your network localhost address. (add port used, if needed)
	a. Found in onCreate function, within the button listener
2. Start XAMPP, and start Apache and mySQL
	a. Database should be created already.
	b. Create database models, if uncreated.
3. Run emulator and wait for application to open itself
4. Enter SID in editBox (search box)
5. Click button (GRADUATE?)
6. Close emulator when finished.
